# guIDE

Local-first IDE with integrated AI (Electron).

## Releases

Installers are published on [GitHub Releases](https://github.com/FileShot/guide-3.0/releases). See [docs/RELEASE_HISTORY.md](docs/RELEASE_HISTORY.md) for archived release notes from older versions.

## Build

Requires Node.js 20+, npm, and platform-specific tooling for native modules. See `.github/workflows/build.yml` for CI build steps.
